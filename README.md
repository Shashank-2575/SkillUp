# SkillUp

### AI-Driven Career Guidance Website

## 📌 Project Overview

**SkillUp** is an AI-Driven career guidance platform designed to help users discover suitable career paths based on their interests, strengths, and preferences.

The system analyzes user input and provides intelligent recommendations along with structured guidance and future direction.

This project was developed as part of the Innovative Assignment submission.
